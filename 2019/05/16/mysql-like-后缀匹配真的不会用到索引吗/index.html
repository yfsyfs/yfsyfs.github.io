mysql like 后缀匹配真的不会用到索引吗

缘起

业余草大哥问（kao）了我一个问题

%y 是用索引了吗?

起初不假思索，答曰: 不会，因为是后缀匹配. 但是上机实操之后, 傻了眼.

会用到index类型索引的!!!

但是稍后查了一下《高性能msql》 又放了心.

explain计划中的index类型的策略和全表扫描一样。只是扫描表的时候按照索引次序进行而不是行。主要优点就是避免了排序, 但是开销仍然非常大。一般保证查询至少达到range级别，最好能达到ref。
